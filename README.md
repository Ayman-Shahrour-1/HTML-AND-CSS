The centerpiece of this repository is a responsive cinema application that demonstrates structured layout and component integration.

### **Core Deliverables:**
* **Intelligent Navigation:** A fixed Bootstrap navbar with **HTML Bookmarks**. Links for `Home`, `About`, `Showtimes`, and `Deals` jump to specific sections of the page without reloading.
* **Dynamic Jumbotron:** A high-visibility hero section acting as a portal for the cinema's branding.
* **Themed Card Deck:** A responsive grid displaying three major titles:
    * **Kong: Skull Island**
    * **Ghost in the Shell**
    * **Beauty and the Beast**
* **Interactive Showtimes:** Each movie card utilizes **Info-colored Pill Badges** (Bootstrap `badge-pill`) to display six distinct showtimes, from 12:00 p.m. to 9:30 p.m.
